# 📱 JalanJakarta — 安卓安装完整指南

---

## 方法一：电脑编译安装（推荐，100%成功）

### 第一步：安装必要工具

#### 安装 Flutter SDK
1. 打开 https://flutter.dev/docs/get-started/install/windows （Windows）
   或 https://flutter.dev/docs/get-started/install/macos （Mac）
2. 下载 Flutter SDK，解压到 `C:\flutter`（Windows）或 `~/flutter`（Mac）
3. 添加到环境变量 PATH：
   - Windows：搜索"环境变量" → Path → 添加 `C:\flutter\bin`
   - Mac：在 `~/.zshrc` 添加 `export PATH="$PATH:~/flutter/bin"`

#### 安装 Android Studio
1. 下载：https://developer.android.com/studio
2. 安装时勾选 "Android Virtual Device"
3. 安装完成后打开 → SDK Manager → 安装：
   - Android SDK Platform 33 或 34
   - Android SDK Build-Tools
   - Android Emulator

#### 验证安装
```bash
flutter doctor
```
确保 Flutter、Android toolchain、Android Studio 都显示 ✓

---

### 第二步：解压并配置项目

```bash
# 1. 解压下载的 zip 文件
# 2. 打开终端/命令提示符，进入项目文件夹
cd jalan_jakarta

# 3. 安装所有依赖
flutter pub get
```

---

### 第三步：配置 Google Maps API Key

1. 打开 https://console.cloud.google.com
2. 新建项目 "JalanJakarta"
3. 启用以下 API：
   - Maps SDK for Android ✓
   - Places API ✓
4. 创建 API Key（凭据 → 创建凭据 → API 密钥）
5. 打开项目文件：
   `android/app/src/main/AndroidManifest.xml`
6. 找到这一行并替换：
   ```xml
   android:value="YOUR_GOOGLE_MAPS_API_KEY"
   ```
   改为：
   ```xml
   android:value="AIzaSy你的真实Key填这里"
   ```

---

### 第四步：编译安装到手机

#### 方式A：直接安装到 Android 手机（最简单）

```bash
# 1. 手机开启"开发者模式"：
#    设置 → 关于手机 → 连续点击"版本号"7次
#    → 返回 → 开发者选项 → 开启"USB调试"

# 2. 用 USB 数据线连接电脑

# 3. 检查是否识别到手机
flutter devices

# 4. 直接安装运行
flutter run

# 或指定设备安装
flutter run -d [设备ID]
```

#### 方式B：生成 APK 文件（可分享给别人）

```bash
# 生成 debug APK（直接安装，无需签名）
flutter build apk --debug

# APK 文件位置：
# build/app/outputs/flutter-apk/app-debug.apk

# 生成 release APK（正式版，需要签名）
flutter build apk --release

# APK 位置：
# build/app/outputs/flutter-apk/app-release.apk
```

#### 方式C：生成 AAB（上架 Google Play 专用）

```bash
flutter build appbundle --release
# 文件位置：build/app/outputs/bundle/release/app-release.aab
```

---

### 第五步：安装 APK 到手机

**方法1：ADB 命令**
```bash
adb install build/app/outputs/flutter-apk/app-debug.apk
```

**方法2：传输文件**
1. 将 APK 文件复制到手机
2. 手机上用文件管理器找到 APK
3. 点击安装
4. 如提示"未知来源"→ 设置 → 安全 → 允许未知来源

## 方法二：使用 GitHub Actions 云端编译（无需强电脑）

如果电脑配置较低，可以用 GitHub 免费云端编译：

1. 注册 GitHub 账号（免费）
2. 新建仓库，上传项目代码
3. 在项目根目录创建文件：
   `.github/workflows/build-apk.yml`

```yaml
name: Build APK
on: [push]
jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - uses: subosito/flutter-action@v2
        with:
          flutter-version: '3.19.0'
      - run: flutter pub get
      - run: flutter build apk --release
      - uses: actions/upload-artifact@v3
        with:
          name: JalanJakarta-APK
          path: build/app/outputs/flutter-apk/app-release.apk
```

4. Push 代码后，GitHub 自动编译
5. 在 Actions 页面下载 APK 文件

## 上架 Google Play 商城

### 准备工作
1. 注册开发者账号：https://play.google.com/console
   费用：**一次性 $25 美元**（约 400 人民币）

2. 创建签名 Keystore：
```bash
keytool -genkey -v \
  -keystore jalan_jakarta.keystore \
  -alias jalanjakarta \
  -keyalg RSA \
  -keysize 2048 \
  -validity 10000
```

3. 配置签名 `android/key.properties`：
```
storePassword=你设的密码
keyPassword=你设的密码
keyAlias=jalanjakarta
storeFile=../jalan_jakarta.keystore
```

4. 修改 `android/app/build.gradle` 添加签名配置（已在文件里预留位置）

5. 编译 AAB：
```bash
flutter build appbundle --release
```

6. 在 Google Play Console：
   - 创建应用
   - 上传 AAB 文件
   - 填写应用信息（截图、描述）
   - 提交审核（通常 1-3 天）

## 常见问题

| 问题 | 解决方法 |
|------|---------|
| `flutter: command not found` | 重新添加 PATH 环境变量后重启终端 |
| `No devices found` | 检查 USB 调试是否开启，换数据线 |
| 地图不显示 | 检查 Google Maps API Key 是否正确填写 |
| 安装时提示"解析包时出现问题" | 手机 Android 版本太低（需要 5.0+） |
| `Gradle build failed` | 运行 `flutter clean` 再重试 |
| 网络图片不加载 | 检查手机网络连接 |
